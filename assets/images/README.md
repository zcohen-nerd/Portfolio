# Placeholder Images

This directory contains placeholder images for the portfolio site, used during development and testing.

## Image Guidelines

### Hero Images (16:9 ratio)
- **Size**: 1200x675px minimum
- **Format**: PNG or JPG
- **Use**: Main project banners, portfolio hero

### Project Images (4:3 ratio)
- **Size**: 800x600px minimum
- **Format**: PNG or JPG
- **Use**: Project thumbnails, process photos

### Diagram Images
- **Size**: Variable, but maintain readability
- **Format**: PNG for line art, JPG for photos
- **Use**: Technical diagrams, flowcharts, schematics

## Naming Convention
- Use kebab-case: `project-name-hero.png`
- Include descriptive suffix: `-hero`, `-diagram`, `-process`
- Keep filenames under 50 characters
