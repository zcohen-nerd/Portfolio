# Autonomous Surfer Fleet

![Project image placeholder](/assets/images/placeholder.png)

## Summary

High-level description of multi-unit autonomous platforms and mission goals.

## Technical Overview

- Platform: propulsion, hull, power distribution, waterproofing.
- Electronics: PCBs, comms, GNSS/IMU, telemetry.
- Autonomy: state estimation, behaviors, base station.
- Operations: charging, maintenance, field procedures.

## Design Process

1. Systems requirements and environment.
2. Mechanical and electrical co-design.
3. Fleet management and data pipeline.
4. Field testing and iteration loops.

## Tools & Technologies

- CAD/CFD, EDA, embedded RTOS, simulation.
- Telemetry services and analysis scripts.

## Results & Lessons Learned

- Reliability and endurance metrics.
- Communications robustness outcomes.
- Operational best practices.
