# Portfolio Content Accuracy To-Do List

This checklist focuses on credibility and accuracy.

## Priority Actions
- [ ] Standardize role/title wording across all pages.
- [ ] Remove or qualify unsupported metrics and outcomes.
- [ ] Confirm project status labels match actual maturity.
- [ ] Keep claims limited to work that is publicly supportable.
- [ ] Replace placeholders with real artifacts or neutral text.

## Content Hygiene
- [ ] Add context (scope, role, constraints) for each project.
- [ ] Keep teaching outcomes qualitative unless cohort data is available.
- [ ] Keep scholarship entries conservative until metadata is verified.
- [ ] Add periodic content validation dates during updates.
