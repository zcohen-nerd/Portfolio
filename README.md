## About This Portfolio

This repository contains the source for my professional portfolio.

The site documents selected work across:
- electromechanical system integration,
- embedded hardware and control implementation,
- and engineering education/lab development.

I use this portfolio as a **living documentation set** rather than a static résumé. Content is updated as projects mature and as public-shareable artifacts become available.

Live site: https://zcohen-nerd.github.io/Portfolio/
